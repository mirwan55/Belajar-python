## 🚀 Cara Pakai

**1. Clone repo ini:**
```bash
git clone https://github.com/mirwan55/Belajar-python.git
cd Belajar-python
```

**2. Buka Jupyter Lab:**
```bash
jupyter lab
```

**3. Pilih sesi yang ingin dipelajari dan buka file `.ipynb`-nya**

> Pastikan sudah install [Anaconda](https://www.anaconda.com/download) — semua library yang dibutuhkan sudah tersedia.
